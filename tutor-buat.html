Fase 2: Mathematics Engine

Tutorial GeoGebra

Langkah membuat semua unsur balok interaktif:

  1. Badan Balok: Buat slider p, l, t. Masukkan input A=(0,0,0) dan Box(A, p, l, t).
  2. Diagonal: Gunakan Segment antar titik (misal A ke G untuk Diagonal Ruang).
  3. Jaring-jaring: Klik balok, pilih tool Net.
  4. Checkbox: Tambahkan tool Checkbox untuk memunculkan/menyembunyikan unsur.

Cara Integrasi ke HTML:

<!-- Gunakan Wrapper agar Responsif -->
<div class="ratio ratio-16x9">
  <iframe src="URL_GEOGEBRA_MU"></iframe>
</div>
Fase 3: Smart Development

Strategi Prompt AI

Jika stuck saat coding fitur (seperti kuis atau modal), gunakan prompt ini:

"Halo Gemini, buatkan kode HTML/CSS/JS lengkap untuk halaman KUIS di proyek GeoSMART. 
Kriterianya: Bootstrap 5, Dark Mode, Aksen Biru Muda (#38bdf8), 
5 soal geometri balok, dan tombol kembali ke dashboard.html. 
Jangan gunakan simbol bintang (*) agar mudah saya copy."
Tips: Selalu cek path gambar. AI sering menulis src="image.jpg", pastikan ganti ke src="assets/images/nama_file.png".
Fase 4: Deployment

Alur Git Bash

Rutinitas Update:

git add .
git commit -m "Update Fitur X"
git pull origin main
git push origin main

Jika Ada Error (Troubleshooting):

  • Identitas: git config --global user.email "mail@mu.com"
  • Beda Versi: git pull origin main --rebase
  • Cek File: git status (Pastikan hijau sebelum push)
Live Link: Repository > Settings > Pages > Branch: Main > Save. Website-mu aktif dalam 1 menit!
